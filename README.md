# Wanted Manager

## 프로젝트 설명

원티드(`https://www.wanted.co.kr`)에서 조건에 맞는 채용 공고를 내 화면에서만 숨기는 크롬 확장 프로그램(Manifest V3)입니다.
네트워크 차단/변조 없이 DOM 조작만 사용합니다.

## 주요 기능

- 회사명/제목/공고 링크(ID) 조건 기반 숨김
- 매칭 방식 `AND` / `OR`
- 모든 규칙은 `hide` 방식으로 동작
- 카드 우측 상단 `이 공고 숨기기` 버튼으로 빠른 숨김 + ID 규칙 자동 저장
- 메인(`/`) 및 `/wdlist` 동적 로딩/무한 스크롤 환경에서 지속 적용
- `/wdlist`의 공고 카드 및 `적극 채용 중인 회사` 카드 모두 규칙 적용
- 팝업에서:
  - 규칙 추가
  - 가린 공고 목록 확인 및 `다시 활성화`
  - 규칙 목록 확인/활성 토글/삭제

## 설치 및 빌드

1. 의존성 설치

```bash
npm i
```

2. 빌드

```bash
npm run build
```

3. Chrome에 로드
- `chrome://extensions` 접속
- 개발자 모드 ON
- "압축해제된 확장 프로그램을 로드" 클릭
- 이 프로젝트 루트 폴더 선택

## 사용 방법

1. 원티드 페이지(메인 또는 `/wdlist`)에서 확장 팝업을 엽니다.
2. 규칙 추가에서 회사명/제목/링크(ID) 중 하나 이상 입력 후 규칙을 저장합니다.
3. 조건에 맞는 카드는 자동으로 숨겨집니다.
4. 카드에서 `이 공고 숨기기` 버튼을 누르면 해당 공고 ID 규칙이 자동 생성됩니다.
5. 팝업 `가린 공고 목록`에서 `다시 활성화` 버튼으로 즉시 복구할 수 있습니다.
6. 팝업 `규칙 목록`에서 규칙 활성/비활성 또는 삭제를 관리합니다.
